# Lead Generation Agent Demo

## How to Run
1.  **Backend**: Open a terminal, navigation to `server/` and run `uvicorn main:app --reload`.
2.  **Frontend**: Open a terminal, navigate to `client/` and run `npm run dev`.
3.  **View**: Open your browser to `http://localhost:5173`.

## Deliverables
- **Ranker Logic**: Located in `server/ranker.py` (Rule-based weights).
- **Mock Data**: Located in `server/crawler.py` (High fidelity profiles).
- **Dashboard**: Located in `client/src/components/Dashboard.jsx` (Premium UI).
